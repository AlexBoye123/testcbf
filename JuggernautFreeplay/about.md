# Juggernaut Freeplay (JFP) v1.7.0

Juggernaut Freeplay generates procedural random wave challenge corridors directly inside Geometry Dash.

### Features
- **Juggernaut Corridor Rule**: Customizable procedural corridor geometry.
- **Dynamic Portals**: Random speed, size, and gravity flips.
- **Anti-Spam / ZigLimit**: Automatically prevents un-passable tight bottlenecks.
- **Full Parity for GD 2.2081 & Geode v5.8.2**.
